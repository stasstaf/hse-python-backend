# Homework 1 - ASGI

## Установка и запуск проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/stasstaf/hse-python-backend
cd hw1
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```


### 3. Запуск приложения

```bash
uvicorn main:app
```

### 4. Запуск тестов

```bash
pytest tests/test_homework_1.py
```
